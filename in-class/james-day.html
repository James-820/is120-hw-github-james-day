Game Description

Halo Flashpoint is an engaging tactical miniatures game made by Mantic Games in which players battle each other with Halo themed miniatures. This game features detailed miniatures and intuitive rules that help each match feel like a real Halo battle. Starter sets come with game tokens and terrain that is ready made to slot together and use, making setup practically painless. Painting the miniatures is encouraged, but they come ready to play in colored plastic.
